# RDBMS (SQL) Data Science Group Project

Analysing structural differences in the US and German labour markets using
World Bank World Development Indicators data, a SQLite relational database,
and Python for data fetching, querying and visualisation.

## Setup

### 1. Clone the repository
```bash
git clone https://github.com/username/ds-sql-project.git
cd ds-sql-project
```

### 2. Install dependencies

Python 3.8 or later is required.
```bash
pip install -r requirements.txt
```

The requirements.txt contains:
```
pandas
matplotlib
requests
```

### 3. Create the database schema

Run the schema file using SQLite to create all tables, indexes and insert
the static economic shocks reference data.
```bash
sqlite3 labour_market.db < schema/schema.sql
```

### 4. Fetch the data

Run the fetch script to pull all indicators from the World Bank API and
populate the database. This requires an internet connection and may take
a few minutes as it fetches data for 200+ countries across 30+ years.
```bash
python data/fetch_worldbank.py
```

Once complete, the following tables will be populated in `labour_market.db`:

| Table | Contents |
|---|---|
| `countries` | Country metadata including region and income group |
| `labour_participation` | Overall, male and female participation rates by year |
| `age_participation` | Youth participation rates (15-24) by gender and year |
| `sector_employment` | Agriculture, Industry and Services employment shares by year |
| `labour_shocks` | Static reference data for known economic events |

### 5. Run the analysis scripts

Each script queries the database and saves a chart to `analysis/graphs/`.
Run them from the root of the repository so the database path resolves correctly.
```bash
python analysis/python/question_1a.py
python analysis/python/question_1b.py
python analysis/python/question_3a.py
python analysis/python/question_3b.py
python analysis/python/question_5a.py
python analysis/python/question_5b.py
python analysis/python/question_5c.py
```

## Research Questions

| Question | Script | Description |
|---|---|---|
| Q1a | question_1a.py | How have overall participation rates evolved in the US and Germany since 1990? |
| Q1b | question_1b.py | What is the year-by-year participation rate gap between the two countries? |
| Q3a | question_3a.py | How do youth participation rates compare across snapshot years? |
| Q3b | question_3b.py | Which gender and age group shows the largest difference between countries? |
| Q5a | question_5a.py | Do the three sector employment shares sum to 100 for every country and year? |
| Q5b | question_5b.py | How have sector employment shares shifted over time in each country? |
| Q5c | question_5c.py | Which sectors show the largest employment share difference in the most recent year? |

## Data Source

All data is sourced from the [World Bank World Development Indicators](https://databank.worldbank.org/source/world-development-indicators),
drawn from ILO modelled estimates. The data is open access and consistent
across both countries for the full period from 1990 to 2023.

| Indicator Code | Description |
|---|---|
| SL.TLF.CACT.ZS | Labour force participation rate, total |
| SL.TLF.CACT.MA.ZS | Labour force participation rate, male |
| SL.TLF.CACT.FE.ZS | Labour force participation rate, female |
| SL.TLF.ACTI.1524.ZS | Youth participation rate, ages 15-24, total |
| SL.TLF.ACTI.1524.MA.ZS | Youth participation rate, ages 15-24, male |
| SL.TLF.ACTI.1524.FE.ZS | Youth participation rate, ages 15-24, female |
| SL.AGR.EMPL.ZS | Employment in agriculture (% of total) |
| SL.IND.EMPL.ZS | Employment in industry (% of total) |
| SL.SRV.EMPL.ZS | Employment in services (% of total) |

## Notes

- `labour_market.db` is excluded from version control via `.gitignore` as it
  is generated locally by running the setup steps above
- The `analysis/graphs/` directory contains the PNG chart outputs and is
  tracked in git so charts are visible on GitHub without running the scripts
- Age group coverage is limited to the 15-24 group as broader age band
  indicators (25-54, 55-64, 65+) returned no data from the World Bank API
