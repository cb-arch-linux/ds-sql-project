## Setup

### 1. Clone the repo
```bash
git clone https://github.com/username/ds-sql-project.git
cd ds-sql-project
```

### 2. Install the database
```bash
# PostgreSQL (macOS)
brew install sqlite3
brew services start sqlite3
```

## Project Structure
```
ds-sql-project/
├── data/         # Python script to fetch data

├── queries/      # All SQL query files
```
